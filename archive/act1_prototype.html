Pair the Manifest

Halo dumps the corrupted manifest array onto your datapad. The contract demands you find the indices of the two values that sum exactly to the target checksum. Return them as a comma-separated pair, smaller index first. The diner's espresso machine hisses behind you. Two more headlights have just pulled into the lot.

nums   = [11, 2, 15, 7, 4, -3]
target = 9

// Find indices i, j such that nums[i] + nums[j] = target.
// Each input has exactly one solution. Don't use the same index twice.
show approach › 
// O(n) hash-map approach — the one interviewers want to see.
def two_sum(nums, target):
    seen = {}                    # value -> index
    for i, x in enumerate(nums):
        need = target - x
        if need in seen:
            return [seen[need], i]
        seen[x] = i
// Brute force is O(n^2) and will get you a polite "can you do better?"
DS / CS CAREER NOTE This is LeetCode #1: Two Sum — literally the most-asked screening question in the industry. The lesson is bigger than the problem: a hash map turns an O(n²) double-loop into an O(n) single pass. That same trick — "trade memory for time, look up instead of scan" — is the heart of indexing in databases, caching in distributed systems, and feature lookups in ML pipelines. You will use this pattern for the rest of your career.

Merge the Convoys

You have two lists of timestamps (in seconds since midnight), each already sorted. Merge them into a single sorted list. Submit as comma-separated values.

convoy_A = [8048, 8730, 8941, 9251, 9550]   // sorted ascending
convoy_B = [8302, 8804, 9704, 9809]         // sorted ascending

// Output: a single ascending list combining both.
show approach › 
// Two-pointer merge — O(n + m), no extra sort needed.
def merge(a, b):
    i, j, out = 0, 0, []
    while i < len(a) and j < len(b):
        if a[i] <= b[j]:
            out.append(a[i]); i += 1
        else:
            out.append(b[j]); j += 1
    out.extend(a[i:]); out.extend(b[j:])
    return out
DS / CS CAREER NOTE This is LeetCode #21: Merge Two Sorted Lists, and the technique — a two-pointer linear merge — is the engine inside merge sort, the merge step in MapReduce, and the way every database stitches sorted index scans into a final result set. The "linear, no resort" instinct is what separates someone who has read about algorithms from someone who has used them.

// FIELD ENCOUNTER 03 — PACKET VALIDATION

Is It a Person?

Below is the bracket envelope of the unsigned packet. A real handshake has every opener matched to its right closer in the right order. A bot probe almost always gets it wrong. Decide whether the string is balanced.

packet_envelope = "{[()]([]{})}{[]}"

// Valid if every '(' '[' '{' has a matching ')' ']' '}'
// in correctly nested order.
// Return: VALID  or  INVALID
show approach › 
// Stack-based, O(n).
def is_valid(s):
    pairs = {')': '(', ']': '[', '}': '{'}
    stack = []
    for ch in s:
        if ch in '([{':
            stack.append(ch)
        else:
            if not stack or stack.pop() != pairs[ch]:
                return False
    return not stack
DS / CS CAREER NOTE This is LeetCode #20: Valid Parentheses. The deeper lesson is that a stack — LIFO — is the natural data structure any time you have nested or recursive structure to validate: JSON parsers, HTML linters, compiler syntax checks, expression evaluators, depth-first traversals. When you reach for recursion, you're often reaching for a stack underneath. Recognizing the shape of a problem ("this nests → use a stack") is the meta-skill interviewers test for.
